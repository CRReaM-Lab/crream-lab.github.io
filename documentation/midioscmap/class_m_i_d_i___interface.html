◆ getDefault()

MIDI_Interface * MIDI_Interface::getDefault ( )
static

Return the default MIDI interface. If the default MIDI interface was configured explicitly using setAsDefault(), that interface is returned. If it wasn't set, or if that MIDI interface no longer exists, this function returns the newest MIDI interface, the one that was constructed most recently. If no MIDI interfaces exist, nullptr is returned.
