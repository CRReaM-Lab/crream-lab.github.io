◆ initiate_security

bool BLESettings::initiate_security = false

Set to true if you want the Arduino to always initiate the Bluetooth bonding or secure connection. As a result, it will show up as a "paired" device on your computer/phone/tablet. If set to false, security is still supported, but the central device should take the initiative.
