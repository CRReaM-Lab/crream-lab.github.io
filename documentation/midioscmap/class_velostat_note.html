Detailed Description

A class for sending Polyphonic Aftertouch messages based on an analog input from a Velostat sensor.

This class reads analog input, applies filtering, and sends MIDI Polyphonic Aftertouch (Poly Pressure) messages, making it ideal for expressive MIDI controllers.
